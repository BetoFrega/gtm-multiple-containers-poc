Hello world! Click anywhere and check the browser console. Only one GTM is going to be triggered depending on which one executes first.

Example page demonstrating what happens when GTM is implemented using two differently named data layers.

From Google's dev guide:

"Note that you can use only a single common data layer for all Google Tag Manager containers on a page because using more than one data layer can cause some triggers to stop working and could have other implications. So don't rename the data layer for a subset of containers on the page. You can, if necessary, rename the data layer for all containers on the page."

More information can be found in https://developers.google.com/tag-manager/devguide#multiple-containers
